No hard card borders. Tiles float via ambient glow shadows, with colored top-stroke accents on key tiles. The condition split is redesigned as a dominant number + pill tags + proportion bar instead of three equal columns. Sets gain inline bars. Charts are more readable.

Recommended Glass Architecture

Double-bezel nested cards: a thin outer shell with a hairline border wraps a distinct inner core with an inner top-highlight. Each tile type reads differently. Sets show proportion bars per item. A tiny eyebrow label appears once per tile (not on every section of the page). The amber alert tile gets a warm glass treatment. The most sophisticated of the three approaches.

Option A — Ambient Depth
Biggest visual gain from removing the flat border ring. The condition tile redesign (dominant number + pills + bar) is the standout improvement here. Closest feel to the current layout, easiest path if you want incremental change.
Option B — Glass Architecture
Recommended. The nested shell gives real physical depth without visual noise. Each tile type reads distinctly. The per-set proportion bars make the sets tile much more scannable. Most premium feel with the least code change from the current Tailwind setup.
Option C — Editorial Flush
Highest typographic contrast and information clarity. Works best if the rest of the watchlist page also moves toward a more editorial, borderless direction. The biggest departure from the current design. Consider this if you plan a broader Dashboard rethink.
